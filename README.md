DD-Mon
======

Fetch data from your favorite [dd-wrt](http://www.dd-wrt.com/site/index) box.

For now, it's a POC, soon, it will push data to [graphite](http://graphite.wikidot.com/)

Test it
-------

    export DD_URL=http://root:toto@192.168.11.1
    ./client.py
